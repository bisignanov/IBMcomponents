## Overview

The FullPageError component renders a full-page error message to inform users
about encountered issues. It's useful for providing clear feedback and guiding
users on what to do next when errors occur.\
[Design Guidance](https://pages.github.ibm.com/cdai-design/pal/patterns/full-page-errors/usage 'fullPageError design guidance')
