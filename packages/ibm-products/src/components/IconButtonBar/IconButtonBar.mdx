## Overview

{/* TODO: Overview. */}

Quickly reduce the number of visible buttons in a toolbar by moving them into an
overflow menu.

If the item's position in the toolbar is less than this component's `length`
prop, it will be rendered as a button. If it's greater than the `length` prop,
it will be rendered as an overflow menu item.

```jsx

  overflowMenuProps={{
    align: 'bottom', // tooltip position
    flipped: true, // menu to the left of the [⋮]
    direction: 'top', // menu above the [⋮]
    iconDescription: 'More', // tooltip text
  }}
>
```
