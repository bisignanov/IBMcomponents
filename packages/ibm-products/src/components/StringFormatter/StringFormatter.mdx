## Overview

StringFormatter allows for truncating text while displaying a tooltip overlay on
hover or focus with the entirety of the provided copy.

## Code sample

```jsx
<StringFormatter
  className="myStringFormatter"
  lines={1}
  truncate={true}
  value="Hello world"
  width="200px"
  tooltipDirection="bottom-left"
/>
```
